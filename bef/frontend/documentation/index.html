CSS Files

all CSS files are located at `css` directory. I will explain them below:

  1. animate.css: used to define custom animations for modern browsers. Read `How to animate elements...` section to learn more.
  2. Bootstrap.css, Bootstrap.min.css: this is Sleek, intuitive, and powerful mobile-first front-end framework for faster and easier web development.
  3. We used latest release of bootstrap framework for this template, version 3 release candiate 2.
  4. nouislider.fox.css: used for jQuery range slider plugin.
  5. responsive.css: our custom responsive rules and media queries are here.
  6. style.css: our main stylesheet.
  7. css/rev-slider/style.css: revolotion slider stylesheet.
  8. css/flexslider/flexslider.css: Flex slider stylesheet.

Also there are two other directory which contain sliders related files.

 

Javascript Files

items 1 to 7 are included on every pages and items 8 to end are used in particular pages.

  1. jquery.min.js: minified jQuery version 1.8.3
  2. bootstrap.min.js: twitter bootstrap framework jQuery plugins version 3 RC2
  3. modernizr.min.js: used for browser feature detection and also HTML 5 shiv for IE8
  4. responde.min.js: used to enable responsive behavior for IE8
  5. jquery.tweet.js: jQuery Twitter API plugin to load latest tweet feeds
  6. package.min.js: couple of jQuery plugins that we used all gathered and compressed in a file
  7. script.js: custom javascript functions that running when DOM is ready
  8. jquery.carouFredSel.min.js: used for products carousel and any other carousels
  9. jquery.nouislider.min.js: jQuery plugin for range slider (used in products page at sidebar)
  10. jquery.zoom-min.js: jQuery plugin that lets we zoom images live (used in single product details page)
  11. jquery.mixitup.min.js: jQuery plugin to sort/filter and switch layouts mode of products (used in products page)
  12. jquery.flexslider.min.js: jQuery plugin for reponsive content slideshow. version 2.0
  13. ddaccordion.js: jQuery plugin for accordion menus.

How to animate elements...

You may notice we have used some nice animation effects in this template. here is demo page of animations that you can use in this template. these animations are only supported by modern browsers including IE10+, Firefox, Google chrome, Safari, Opera. animations will be run when animated object displayed in viewport.

To add animation to any element, follow these steps:

  1. add class `animated` to element.
  2. add attribute data-animation and set your desired animation

i.e. <div class="animated" data-animation="fadeIn"></div>

How to configure twitter API

Open this file from your downloaded package `/Twitter/index.php`, Then set your API keys (You need to create an twitter app and generate API key at here http://dev.twitter.com/apps ):

// Your Twitter App Consumer Key
private $consumer_key = '';
 
// Your Twitter App Consumer Secret
private $consumer_secret = '';
 
// Your Twitter App Access Token
private $user_token = '';
 
// Your Twitter App Access Token Secret
private $user_secret = '';
 
Leave other configs as they are, or edit them by your risk.

Contact us page

Google map

to edit google map, open contact.php or contact.html with your editor and search for `contact-map`. you will find this code block:

<div id="contact-map" class="google-map" data-lat="" data-lng="" data-mlat="" data-mlng=""></div>

as you can see, there are 4 attributres that by set proper values you can edit google map marker and google map area to what you need.

to find proper latitude and longitude of your place, open https://maps.google.com and find your place. then right click on map and choose `What's here?`. latitude and longitude values are now available in search input. 

for example, envato address on google map cordination is : -37.817186, 144.964986

so now we just set the values:

  • data-lat=""         => data-lat="-37.817186"
  • data-lng=""        => data-lng="144.964986"

to place marker on map set data-mlat and data-mlng values like above. note that you can add multiple map marker just seprate them by comma like this:

data-mlat="-37.817186,-37.818186,...."

data-mlng="144.964986,144.963986,..."

Contact E-mail form

to configure contact form and destination email adderss go to `includes/phpmailer/` and open contact.php with your editor and configure it.

$host = 'mail.yourdomain.net'; // Set the hostname of the mail server
$port = 25; // Set the SMTP port number - likely to be 25, 465 or 587
$username = 'noreply@themina.net'; // Username to use for SMTP authentication
$password = '#########'; // Password to use for SMTP authentication
 
$config['to] and $config['to_name'] defines to what email address and name should contact form data be send.
$config['from'] and $config['from_name'] defines email address and name to send Thanks email
$config['thanks_subject'] and $config['thanks_body'] define thanks message to send to user after success contact email submit.
